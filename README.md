Website resmi toko **Aneka Citra Computer** yang berlokasi di area *Jambu 2*.  
Website ini berfungsi sebagai platform e-commerce sederhana untuk **penjualan perangkat elektronik** seperti **laptop, printer, dan aksesoris komputer**.

---

## Tentang Project

Aneka Citra Computer adalah toko yang menyediakan berbagai perangkat elektronik, khususnya **komputer, laptop, printer, dan perangkat pendukung lainnya** untuk kebutuhan personal maupun bisnis.

Website ini dirancang sebagai **toko online sederhana** agar pengunjung dapat dengan mudah melihat katalog produk, informasi toko, serta gambaran layanan yang tersedia melalui browser.

Pengembangan website dilakukan menggunakan teknologi web dasar tanpa backend kompleks atau database, dengan desain yang **responsive** dan mudah digunakan.

---

## Fitur Utama

Fitur yang tersedia dalam website ini meliputi:

- Halaman Beranda (Homepage)  
  Menampilkan informasi singkat tentang toko serta produk unggulan.

- Katalog Produk  
  Menampilkan daftar produk elektronik seperti laptop, printer, dan aksesoris komputer.

- Deskripsi Produk  
  Informasi singkat mengenai spesifikasi dan detail produk.

- Responsive Design  
  Tampilan website menyesuaikan dengan perangkat desktop maupun mobile.

- Custom CSS Styling  
  Penggunaan CSS untuk tata letak dan tampilan visual website.

- Informasi Kontak dan Lokasi  
  Menyediakan informasi alamat dan kontak toko.

- Jenis jenis

Website ini sedang dikembangkan lebih lanjut dengan penambahan fitur baru.

## Teknologi yang Digunakan

Website ini dibangun menggunakan teknologi berikut:

- HTML5
